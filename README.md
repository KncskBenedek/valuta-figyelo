# Valuta Figyelő
## A futtatáshoz a gyökér mappában a "make run_all" parancs segítségével tudjuk futatni
Ehez szükséges a "node" és a "make"-nek a megléte 
## Ha nincs "make"
### Frontend futtatásacd 
	cd valuta-figyelo
    npm install
    npm run dev 
### Proxy futtatása
	cd valuta-proxy
    npm install
    node server.js
## Frontend
A terminálon kiírja hol érhető el a frontend.
